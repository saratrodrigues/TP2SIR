API Documentation

This API allows you to manage students data with basic CRUD operations.

Base URL

The base URL for the API is:

http://localhost:3000/students

Routes

The following HTTP methods are supported:

1. GET /students

Returns a list of all students.

GET http://localhost:3000/students

Example Response:

[
    { "id": 1, "name": "Alice", "course": "Computer Science", "year": 2 },
    { "id": 2, "name": "Bob", "course": "Mathematics", "year": 1 }
]

2. GET /students/{id}

Returns a specific student by ID.

GET http://localhost:3000/students/1

Example Response:

{ "id": 1, "name": "Alice", "course": "Computer Science", "year": 2 }

3. POST /students

Creates a new student. You need to provide the student's name, course, and year.

POST http://localhost:3000/students

Request Body Example:

{
    "name": "John",
    "course": "Physics",
    "year": 3
}

Example Response:

{
    "id": 3,
    "name": "John",
    "course": "Physics",
    "year": 3
}

4. PUT /students/{id}

Updates a student by ID. You can update the name, course, or year of a student.

PUT http://localhost:3000/students/1

Request Body Example:

{
    "name": "Alice Johnson",
    "course": "Computer Science",
    "year": 3
}

Example Response:

{
    "id": 1,
    "name": "Alice Johnson",
    "course": "Mathematics",
    "year": 1
}

5. DELETE /students/{id}

Deletes a student by ID.

DELETE http://localhost:3000/students/1

Example Response:

{ "id": 1, "name": "Alice", "course": "Computer Science", "year": 2 }

Response Codes

The following HTTP status codes are used:

  • 200 OK: Request succeeded.
  • 201 Created: Resource created successfully.
  • 400 Bad Request: Invalid request, often due to missing or incorrect data.
  • 404 Not Found: Resource not found.
  • 500 Internal Server Error: A server error occurred.

Usage Example

To add a new student, send a POST request with the following body:

{
    "name": "Alice",
    "course": "Computer Science",
    "year": 2
}

To fetch all students, send a GET request to:

GET http://localhost:3000/students

To update a student's information, send a PUT request with the student's ID and updated data.
